Welcome To Alan's Image Test Site

This is a free, open-source site where I'll be creating a series of images in different formats (e.g. JPG, PNG, WEBP, AVIF) with various quality settings. The goal is to provide an easy way for folks to choose settings that strike a good balance between image quality and file size. The idea came from writing a post about my switching from JPG to AVIF for the images on my sites.

There's not much here yet. I'm just getting started. (If you're here, it probably means I've contacted you to ask about using one of your images in the comparisons. Either that or you read the moving to AVIF post.)

This is prototype work that's been set up by hand. The plan is to automate image generation to create upwards of 50 versions of each test image with incrementally changing settings. Those will go along with an interface to make it easy to select specific versions for comparison.

To give you an idea of how it'll work, here's an example of a single comparisons from the moving to AVIF post.
